# backtest.sh

## Usage

```shell
git clone git@github.com:nanvel/backtestsh.git
cd backtestsh
uv sync
uv run btsh --help
```

Requires ANTHROPIC_API_KEY:
```shell
export ANTHROPIC_API_KEY=...
```
or use `.env` file.
